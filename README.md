# ProjetoFrontEndSemestre1
Repositório para a criação do projeto do primeiro semestre da aula de projeto front-end
Sistema para controle de fila. Parte interessada qualquer profissional que necessite de um agendamento prévio. Só exibe os horários disponíveis. Deve haver uma parte do sistema onde o profissional inclua seus serviços e valores.

Deve ter a parte do cliente e a parte do profissional, onde ele possa aprovar e gerenciar os agendamentos. Verificar a possibilidade do agendamento ser incluído na agenda de compromissos do telefone.

Utilizar o bootstrap como framewaork CSS
Verificar a linguagem de back-end (Raphael Duarte prefere C#)


Área do cliente:
O cliente deverá criar cadastro(nome, login, senha, telefone, e-mail.)

Fazer login
Visualizar serviços disponíveis   
Valores dos serviços 
Horários disponíveis
Realizar agendamento
Cancelar ou reagendar(de acordo com as regras)
Receber confirmação via e-mail ou WhatsApp
Visualizar histórico de agendamento

Área profissional 
Criar cadastro profissional 
Informar serviços e valores 
Definir horários de atendimento 
Aprovar ou recusar agendamentos
Gerenciar fila de espera 
Verificar agenda diária
Receber notificações de novos agendamentos
Controle financeiro
